3 """
4 This is a package I use to handle numerical-model parameters,
5 thus the name. However, it should be useful otherwise too.
6 It has two main features:
7 - keyword type constructors with default values, and
8 - unpacking and packing of composite types and dicts.
9
10 The macro `@with_kw` which decorates a type definition to
11 allow default values and a keyword constructor:
12
13 ```
14 julia> using Parameters
15
16 julia> @with_kw struct A
17 a::Int = 6
18 b::Float64 = -1.1
19 c::UInt8
20 end
21
22 julia> A(c=4)
23 A
24 a: 6
25 b: -1.1
26 c: 4
27 ```
28
29 Unpacking is done with `@unpack` (`@pack!` is similar):
30 ```
31 struct B
32 a
33 b
34 c
35 end
36 @unpack a, c = B(4,5,6)
37 # is equivalent to
38 BB = B(4,5,6)
39 a = BB.a
40 c = BB.c
41 ```
42 """
